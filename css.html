CSS

What is CSS?

CSS (Cascading Style Sheets) is a language used for describing the presentation of a document written in HTML or XML. It controls the visual layout of web pages, including colors, fonts, and spacing.

Getting Started with CSS

1. Inline CSS

Inline CSS is applied directly within the HTML element using the style attribute. While simple, it's not recommended for larger projects due to maintenance issues. 

<h1 style="color: blue;">Hello, CSS!</h1>

2. Internal CSS

Internal CSS is defined within the <style> tag in the head section of an HTML document. It's useful for smaller projects where CSS rules are limited. 

<style>
    h1 {
        color: blue;
    }
</style>

3. External CSS

External CSS is the recommended method for styling web pages. It involves linking an external stylesheet using the <link> tag in the head section, promoting code reusability and easier maintenance. 

<link rel="stylesheet" href="style.css">

CSS Selectors

CSS selectors are patterns used to select the element(s) you want to style. Here are a few common selectors:

  • Element Selector: Selects elements based on their tag name (e.g., h1, p).
  • Class Selector: Selects elements with a specific class attribute (e.g., .btn).
  • ID Selector: Selects a single element with a specific id attribute (e.g., #header).

Fonts & Colors

CSS provides extensive options for styling fonts and colors. You can specify font families, colors using predefined names, RGB, HEX, HSL values, etc. 

h1 {
    font-family: 'Roboto', sans-serif;
    color: #4CAF50;
}
