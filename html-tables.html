HTML Tables

Welcome to the HTML Tables course! In this course, you will learn how to create and style tables using HTML.

What are HTML Tables?

HTML tables are used to display data in rows and columns. They are essential for organizing and presenting structured information on web pages.

Basic Structure of an HTML Table

An HTML table is defined using the <table> tag. Tables consist of rows (<tr>), which contain table data cells (<td>) or table header cells (<th>).

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>30</td>
    </tr>
</table>

Table Elements and Attributes

HTML provides several elements and attributes for creating and formatting tables:

  • <table>: Defines a table.
  • <tr>: Defines a row in a table.
  • <th>: Defines a header cell in a table.
  • <td>: Defines a data cell in a table.
  • Attributes such as colspan, rowspan, and border can be used to control table layout and appearance.

Styling HTML Tables

Tables can be styled using CSS to enhance their appearance and improve readability. CSS properties like border, padding, margin, and background-color can be applied to table elements.

Advanced Table Techniques

Explore advanced table features such as nested tables, table headers (<thead>), table bodies (<tbody>), table footers (<tfoot>), and table captions (<caption>) to create complex and structured layouts.
